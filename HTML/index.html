  Community Linux is a Personal Class Linux Distribution that boots from a USB drive. It has all the features of hard a drive installation, boasting comparable performance. Community Linux is derived from sources freely available, as provided by the Linux community. Where applicable, ComLin conforms fully to existing Linux standards.

  Community Linux is simple. Just insert the jump drive into a USB slot, and within minutes, a fully functional Graphical User Interface appears.

  Community Linux is created and developed by Peter Glen. It is designed to be an application deployment platform, but can be used by any Linux user, including system administrators, network administrators, enterprise users, managers, core Linux contributors and Linux enthusiasts. Contributions are added to it from around the world.

  Community Linux has numerous advantages, including a speedy USB boot, simple deployment and portability. It allows system integrator to build a hard drive less solution, so Linux can be deployed in harsh environments, and in places it could not boldly go before. It is also a premier platform for system rescue disks. The major application platform for ComLin is intelligent embedded designs like image recognition platform for security systems.

  Community Linux is easy to install. Just copy the image data to a new jump drive, and execute the boot install script(s). Insert that jump drive into a USB slot on the target machine, and boot. (Make sure to set the BIOS to boot from jump drive) The distro will boot into its graphical user interface without intervention. Initially, the distro runs as 'user', so make sure your first action is to change the passwords. Please read the installation section for more info.

  Community Linux boots directly from the USB drive, and uses the drive's free space to store your data. It does not need any hard drives, and will work with most commonly available hardware. During bootup it will prompt you at different stages of boot for a key press. (with a timeout) Left alone, the distro will continue booting, but by pressing the enter key, the boot process can be suspended to a shell. This can be used as a troubleshooting tool. See help for further details.

  Community Linux includes most mainstream applications, including FireFox, Thunderbird, Open Office, Terminal, Secure Shell, File Manager ... etc. A host of other tools and utilities are installed by default, including the full 'C' compiler suit to be able to build programs.

  Community Linux is secure. We included a pre-configured encrypted folder, all you need to do is change the encryption password to your personal preference. Make sure you remember the password, as no recovery is passible, and the data is encrypted with an industrial strength encryption.

  The parent distribution is CentOS 6.3, and the repositories are configured and enabled. Overall there are about ten thousand packages available from the core distro. Make sure to read the disclaimer on the package you wish to install. (for stability, robustness, security information)
